# 🖼 IMGCMP – Image Compressor Web App

A fast and privacy-friendly **file compression web application** that allows users to compress **images, PDFs, and videos directly in the browser**.

⚡ No uploads  
🔒 No server processing  
💻 Everything happens locally in your browser

---

## 🚀 Live Demo

🔗 https://imggcmp.netlify.app/

---

## ✨ Features

- 📉 Image Compression
- 📄 PDF Compression
- 🎬 Video Compression
- 🖼 Image Resizer
- ✨ Image Enhancer
- 🔳 QR Code Generator
- ⚡ Fast client-side processing
- 📱 Fully responsive design
- 💾 Instant download
- 🔒 Privacy friendly (files stay on your device)

## ⚙ Installation

Clone the repository

```bash
git clone https://github.com/GOD694/image-compress.app.git


Move to the project folder

cd image-compress.app

Install dependencies

npm install
▶ Run Development Server
npm run dev

Open in browser

http://localhost:5173
📦 Build for Production
npm run build
🔍 Preview Production Build
npm run preview
